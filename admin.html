Panel de Administración

El administrador crea IDs, les asigna un nombre y registra archivos con la información obligatoria del evento.

ID
Crear identificación Todo ID numérico debe registrarse primero con un nombre.

UP
Cargar archivo Solo el administrador puede asociar archivos a IDs ya creados y con nombre.
+
Archivo obligatorio

Formatos permitidos: MP4, JPG o PNG. MP4: maximo 30 minutos o 1.5 GB.

No se puede guardar ningun archivo sin ID previo, nombre asignado, evento, fecha y descripcion. Los MP4 tienen un limite de 30 minutos o 1.5 GB.

BD
Registros administrados Vista general de IDs, nombres y archivos asociados.

Contenido registrado

Cada ID conserva su nombre incluso si aún no tiene archivos.
